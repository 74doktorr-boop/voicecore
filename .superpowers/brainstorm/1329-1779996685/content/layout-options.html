A — Nueva pestaña en /admin

Se añade un tab "Asistentes" al panel admin existente. Lista de negocios a la izquierda, editor + chat de prueba a la derecha. Todo en un solo sitio.

✅ Pros

  • Un solo sitio, ya tienes el login
  • Consistente con lo que hay

❌ Contras

  • La página admin ya es densa

  • Más espacio para chat y editor

C — Admin expandido (lista + detalle + tabs)

La lista de orgs existente se convierte en un panel maestro-detalle. Clic en una org abre config/prompt/demo en tabs a la derecha, en la misma página.

✅ Pros

  • Todo el flujo en una pantalla
  • Muy rápido cambiar entre orgs

❌ Contras

  • Requiere reescribir más del admin
