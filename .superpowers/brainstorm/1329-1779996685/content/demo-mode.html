A — Chat de texto

Widget de chat en el navegador. Escribes mensajes y el bot responde con el LLM real usando el prompt configurado. Ves exactamente qué diría en cada caso.

✅ Pros

  • Sin latencia de voz
  • Fácil de probar rápido
  • Los negocios lo entienden

❌ Contras

  • No pruebas la voz real

B — Llamada de voz (WebRTC)

Abres una llamada de voz real desde el navegador. Hablas con el bot tal y como lo haría un cliente por teléfono. Prueba completa: voz, velocidad, pausas.

✅ Pros

  • Demo 100% real
  • Pruebas voz + latencia

❌ Contras

  • Necesita mic en el browser
  • Más complejo de implementar

C — Ambos (chat por defecto + voz opcional)

❌ Contras

  • Más UI que construir
