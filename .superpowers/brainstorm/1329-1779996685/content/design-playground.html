Notas del diseño:
  • Panel izquierdo: orgs reales (de Supabase) + bots de prueba (sin org)
  • Org seleccionada → 3 tabs: Config org / Asistente / Demo voz
  • Tab Asistente tiene sub-tabs propios: Básico, Horario, Contenido (sector-specific), Prompt raw, Voz
  • Tab Demo: botón para iniciar llamada de voz en el navegador (mic → STT → LLM → TTS → audio)
  • Portal de negocio: misma estructura de sub-tabs del Asistente, pero sin acceso a Config org ni a la lista de otras orgs
