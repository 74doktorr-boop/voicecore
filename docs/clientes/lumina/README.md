## Configuración del asistente

Config file: `assistants/lumina-estetica.json`

El asistente está configurado para gestionar:
- Reservas de tratamientos (facial, corporal, depilación)
- Información de precios y duración de cada servicio
- Horarios de la terapeuta
- Recordatorios de cita
