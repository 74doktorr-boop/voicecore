## Demo interactiva (playground)

Si el cliente quiere probar hablar con el asistente directamente:

**https://nodeflow.es/demo**

→ Pueden elegir sector, hablar por micrófono y experimentar cómo sería su asistente.

---

## Cómo usar los links

**En WhatsApp:**
```
Te comparto la demo para [su sector]:
https://nodeflow.es/[sector]

Ahí puedes escuchar cómo suena el asistente y ver todo lo que incluye.
¿Lo puedes ver y me dices qué te parece?
```

**Por email:**
Enviar el link de su sector + el link de la demo interactiva.  
Asunto sugerido: "Demo NodeFlow para [Nombre del negocio]"
