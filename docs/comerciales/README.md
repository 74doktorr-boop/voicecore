## Flujo de trabajo rápido

```
1. Buscar leads
   node scripts/buscar-leads.js --sector=dentistas --ciudad=bilbao

2. Abrir Google Sheet → filtrar por ciudad
   → columna wa_link = clic directo a WhatsApp con mensaje personalizado

3. Enviar mensaje de WhatsApp (ya escrito automáticamente)

4. Si responden con interés → enviar link de demo de su sector
   (ver demo-links.md)

5. Llamada de 10 min → guión en pitch-guion.md

6. Si cierran → enviar magic link del portal
   → desde Admin Panel → su organización → "Enviar magic link"
```

## Objeciones más comunes

**"Ya tenemos centralita / contestador automático"**
→ "Una centralita solo desvía llamadas. NodeFlow entiende lo que dice el cliente, gestiona la cita, responde preguntas y anota todo automáticamente. Es un recepcionista real, no un menú de opciones."

**"Es muy caro"**
→ "49€ al mes son menos de 2€ al día. Un no-show evitado ya te lo paga. ¿Cuántas llamadas perdéis cuando estáis atendiendo?"

**"No tenemos tiempo para configurarlo"**
→ "La configuración la hago yo en menos de 24 horas. Solo necesito que me respondas a 10 preguntas sobre tu negocio. Sin instalar nada ni cambiar tu número."

**"¿Y si el cliente nota que es IA?"**
→ "El asistente se presenta como 'el asistente virtual de [tu negocio]'. Los clientes saben que es IA y les parece bien — lo que valoran es que alguien les atiende de inmediato, no que esperen."

**"Probamos con Chatbot X y fue un desastre"**
→ "Los chatbots de texto son otra cosa. NodeFlow atiende llamadas telefónicas reales, habla en castellano y euskera natural, y está entrenado específicamente para [su sector]."
